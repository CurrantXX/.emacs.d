# .emacs.d
My emacs configuration files, for backup

#### Screenshot

![screenshot](https://raw.github.com/gallochoi/.emacs.d/master/screenshot.jpg)

#### Installation
```
$ git clone --recursive https://github.com/Gallonchoi/.emacs.d.git ~/.emacs.d
$ cd .emacs.d/common/helm
$ make
Enjoy
```
